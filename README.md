# SWEEPD

Hi! sweepD is a is a tool for sweeping your project's deprecated functions.

## Installation
    npm install -g sweepd

## Commands

Open cmd in the folder of your project and run the following:

    sweepd js (not implemented yet)
    sweepd php
    sweepd py
